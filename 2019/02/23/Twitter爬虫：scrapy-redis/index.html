scrapy运行过程

首先，scrapy从调度器中取出一个URL用于抓取，然后，scrapy把URL封装成一个Request请求传给downloader
downloader把资源下载下来，封装成Response，爬虫解析Response，解析到Item,交给管道进行处理，若解析出的是URL,就直接把URL交给调度器等待抓取
